# TheDent Redesign

Welcome to the repository for **TheDent Redesign**! This project involves the redesign and development of TheDent website to improve user experience, performance, and aesthetics.

## [Live Demo](https://thedent.com.ua/)

## Overview

TheDent Redesign aims to modernize the website with a fresh look and feel, ensuring it is responsive, accessible, and user-friendly. The project focuses on delivering a seamless experience for visitors while maintaining the brand's identity.

## Features

- Responsive design for all devices
- Improved navigation and user interface
- Optimized for performance and SEO
- Accessibility enhancements
- Integration with modern tools and frameworks

## Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript
- **Backend**: PHP
- **Other Tools**: Slick
